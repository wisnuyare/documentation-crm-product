# User Guide

Welcome to the **CRM Product** user guide. This application allows you to manage customer interactions via WhatsApp using advanced AI agents.

## Getting Started

To start using the system, simply send a message to the WhatsApp bot number.

### Key Features

::: tip Smart Routing
The system automatically routes your request to the correct agent (Booking, Support, or General).
:::

- **Booking Agent**: Handles reservations and availability checks.
- **Transaction Agent**: Manages orders and payments.
- **Support Agent**: Answers general queries and provides assistance.

## Chat Interface

The chat interface is designed to be intuitive. You can speak naturally, and the AI will understand your intent.

### Example Commands

- "I want to book a table for tomorrow at 7 PM"
- "Check the status of my order #123"
- "What are your opening hours?"

## Booking Flow

1. **Initiate**: Ask for a booking.
2. **Details**: Provide date, time, and party size.
3. **Confirmation**: The agent checks availability and confirms the booking.
