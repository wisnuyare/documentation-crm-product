# Tenant Owner Journey

This guide explains how a business owner (Tenant) onboards and manages their CRM instance.

## Onboarding Flow

```mermaid
graph LR
    Register[Sign Up] --> Config[Configure Business Profile]
    Config --> Resources[Add Resources/Products]
    Resources --> Integration[Connect WhatsApp]
    Integration --> Live((Go Live))
```

## Management Activities

### 1. Dashboard Monitoring
View real-time metrics on the **Analytics Dashboard**:
- Active Conversations
- Total Revenue
- Booking Occupancy

### 2. Resource Management
- **Add/Edit Resources**: Update table availability or room status.
- **Update Menu**: Change product prices or descriptions.

### 3. Configuration
- **Business Hours**: Set opening and closing times.
- **Auto-Replies**: Customize welcome messages and fallback responses.
